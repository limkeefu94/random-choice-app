# Changelog

本文件记录 `random-choice-app` 的主要功能进展。格式保持简洁，后续每次合并前都需要更新。

## Unreleased

- 建立正式开发流程文档：`AGENTS.md`、roadmap、release checklist 和任务模板。
- 修复 Vercel Hobby 部署函数数量上限：将 API helper 移出 `api/`，并移除未使用的旧上传 demo。
- 简化个人资料编辑：移除手动头像文字输入，默认文字头像改为根据显示名称或用户名自动生成。
- 新增设置中心骨架：集中整理账号、隐私、内容偏好、应用信息、反馈和未来关联应用入口。
- 新增关联应用占位：设置中心展示未来随心转盘和其他生活小工具共用账号的规划入口。
- 优化手机端个人资料和设置面板：增加底部安全滚动空间，并压缩关联小应用展示。

## v0.6.6 - 世界频道稳定性优化

- 整理世界频道最近更新的版本显示。
- 优化爱心按钮和消息操作区的细节。
- 优化图片预览和裁剪关闭后的状态清理。
- 改善登录、登出后世界频道状态刷新。
- 修正手机端可能出现的按钮拥挤或底部遮挡问题。

## v0.6.5 - 世界频道图片与爱心互动

- 世界频道消息现在可以点爱心。
- 爱心旁边会显示数量，已点过会显示实心爱心。
- 桌面端查看世界频道图片时，图片和文字说明改为左右布局。
- 大图不会在电脑上撑得过高。
- 关闭按钮更容易找到。
- 手机端仍保持适合小屏幕的上下浏览方式。

## v0.6.4 - 世界频道图片浏览优化

- 世界频道的选择图片按钮变得更紧凑。
- 图片提示移到按钮旁边，输入区看起来更整齐。
- 世界频道里的图片现在可以点击放大查看。
- 大图可以用滚轮或双指放大，也会显示原消息。
- 大图可以点击关闭或按 Esc 关闭。

## v0.6.3 - 世界频道体验优化

- 世界频道顶部和输入区变得更紧凑。
- 账号在线状态移到输入区里，不再额外占一张卡。
- 发言提示变得更自然，不再一开始显示字数限制。
- 输入接近字数上限时，才会显示剩余字数。
- 图片选择提示变得更简短。

## v0.6.2 - 图片裁剪交互修复

- 头像裁剪可以拖动调整位置。
- 头像裁剪支持鼠标滚轮缩放和手机双指缩放。
- 修正头像裁剪弹窗显示层级。
- 世界频道发图改为独立裁剪面板，手机上更好操作。
- 登出后会清空本机记录和未发送内容。

## v0.6.1 - 图片裁剪和发送预览

- 头像上传前可以先裁剪，保存后显示更整齐。
- 世界频道发图片前可以先预览。
- 发送图片时可以选择原图、方形、横图或竖图。
- 图片发送前会在前端压缩，减少超大图片上传失败。

## v0.6.0 - 我的主页和我的动态管理

- 点右上角头像会进入我的主页。
- 可以查看自己发过的世界频道文字和图片。
- 自己发过的内容可以编辑文字或移除。
- 世界频道已移除内容会隐藏，后端使用 soft delete。

## v0.5.0 - 设置和版本内容整理

- 点版本号可以查看更新内容。
- 通知用途调整，不再承载完整更新日志。
- 反馈和清除按钮布局优化。
- 设置里移除“隐藏娱乐号码相关内容”开关，后续交给首页自定义功能处理。
- 注册名字示例替换为普通示例。

## 已完成大功能

- GCS 图片上传：支持世界频道图片上传，并预留 signed URL / server upload 流程。
- 账号注册登录：支持基础注册、登录、资料保存和云端账号同步。
- 世界频道：支持登录用户进入公开频道对话。
- 手机端世界频道优化：优化移动端聊天、按钮和顶部栏可用性。
- 娱乐号码随机：补齐多地区号码型随机玩法，并保留娱乐免责声明。
- 购物分类和购物数据：拆分购物数据，补充分类、二级分类、预算和购买提醒。
- 通知优化：缩短通知内容，优化红点和手机端通知面板显示。
- Logo / app icon：加入应用图标、favicon、Apple Touch Icon 和 PWA manifest 基础配置。
- 反馈入口进行中：已预留反馈入口和前端错误上报，用于试用期收集问题。
